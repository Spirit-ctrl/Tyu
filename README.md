Telegram-бот для получения расписания ТИУ

Данный проект представляет собой Telegram-бота, написанного на aiogram 3, который автоматически получает расписание занятий с сайта Тюменского индустриального университета (ТИУ) и отправляет его в Telegram-чат.

Бот использует Selenium для взаимодействия с веб-страницами и BeautifulSoup (bs4) для парсинга HTML-контента. Такой подход позволяет корректно обрабатывать динамически загружаемые страницы сайта университета.

Основная задача бота — автоматизация получения актуального расписания и удобная доставка информации пользователям в Telegram без необходимости вручную заходить на сайт ТИУ.

Основные возможности

Бот заходит на сайт Тюменского индустриального университета, получает данные о расписании занятий и отправляет их в Telegram в удобном текстовом формате. Возможна дальнейшая доработка форматирования сообщений, добавление фильтрации по группе, дате или преподавателю.

Используемые технологии

Проект написан на языке Python. В качестве Telegram-фреймворка используется aiogram версии 3. Для работы с веб-страницами применяется Selenium, а для разбора HTML-кода — BeautifulSoup (bs4).

Установка и запуск

Для работы проекта необходимо наличие установленного Python версии 3.10 или выше, а также установленный браузер и драйвер для Selenium (например, Chrome и ChromeDriver).

После клонирования репозитория необходимо установить зависимости, указанные в файле requirements.txt, настроить переменные окружения (токен Telegram-бота и при необходимости другие параметры), а затем запустить основной файл приложения.

Назначение проекта

Проект создан в учебных и практических целях. Он демонстрирует интеграцию Telegram-бота с парсингом веб-сайтов, работу с динамическим контентом и асинхронный подход в aiogram 3.

Бот не является официальным сервисом Тюменского индустриального университета.
